# Іконки для секції «Переваги» (#why)

Сюди клади іконки, з яких крапки збиратимуть фігуру на слайдах.

## Вимоги
- **Лінійні (line-art)**, контурні — не суцільна заливка.
- Високий контраст: темні лінії на **прозорому або білому** фоні.
- Розмір від **512×512** (більше — детальніше).
- Бажано квадратні, приблизно однакова товщина ліній.
- **SVG найкраще**, PNG теж підійде.

## Маппінг на слайди
| Слайд | Перевага | Файл (приклад назви) |
|------|----------|----------------------|
| 01 | Власне озеленення | `why-01-tree.svg` |
| 02 | Рулонний газон | `why-02-roll.svg` |
| 03 | Автополив | `why-03-sprinkler.svg` |
| 04 | Власна спецтехніка | `why-04-machine.svg` |

Після додавання файлів — підставлю їх замість тимчасових фігур у `initWhyDots`
(якщо SVG — у масив `SHAPES`; якщо PNG — переключу семплер на читання пікселів із зображень).
